# Tech-Focused 3D Interactive Portfolio

## Design Philosophy
A cutting-edge developer portfolio showcasing technical expertise through interactive 3D elements, code aesthetics, and cyberpunk-inspired design. Built with Three.js and Next.js for optimal performance.

---

## Visual Design & Tech Theme

### Color Scheme - Cyberpunk Tech
- **Primary**: Electric Blue (#00f3ff, #0ea5e9)
- **Secondary**: Neon Green (#10b981, #22c55e) 
- **Accent**: Hot Pink/Magenta (#ec4899, #f43f5e)
- **Background**: Dark Matrix (#0a0e27 â†’ #1a1f3a)
- **Code Blocks**: Terminal Black (#0d1117) with syntax highlighting
- **Grid Lines**: Cyan glow (#22d3ee) at 0.3 opacity

### Typography - Developer Focused
- **Headers**: 'Space Grotesk' / 'Rajdhani' - Bold, Angular
- **Body**: 'Inter' - Clean, readable
- **Code**: 'Fira Code' / 'JetBrains Mono' - with ligatures
- **Monospace numbers**: For stats and metrics

## 3D Background Elements - Tech Scene

### Neural Network Particles
- **5000+ connected nodes** forming a digital brain/network
- Lines connect nearby particles (max distance: 150px)
- Particles pulse with data transmission effect
- Mouse interaction creates "signal waves" through the network
- Color: Electric blue â†’ cyan gradient

### Geometric Tech Objects

1. **Rotating Code Lattice** (background)
   - Grid of cubes forming a larger cube structure
   - Binary code texture on surfaces (0s and 1s)
   - Continuous rotation on Y-axis
   - Wireframe edges with neon glow

2. **Floating Microchip/Circuit Board**
   - Low-poly geometric representation
   - Green circuit traces lighting up sequentially
   - Hover effect: Zoom in and show component details
   - Position: Right side, mid-ground

3. **Data Streams**
   - Flowing vertical lines like Matrix rain
   - Binary numbers cascading down
   - Accelerate on scroll/interaction
   - Color: Green â†’ Blue gradient

4. **Holographic Display Panels**
   - Translucent rectangles with hex code patterns
   - Float and rotate slowly
   - Display tech keywords: "React", "Node", "Python", "AI"
   - Glass morphism effect with blur

### Dynamic Lighting
- Scan line effect moving across screen
- Point lights with blue/green glow following cursor
- Ambient occlusion for depth
- Bloom post-processing on bright elements

## Mobile Responsiveness

### Breakpoints
- Desktop: 1920px, 1440px, 1024px
- Tablet: 768px
- Mobile: 640px, 375px

### Mobile Adjustments
- Simplify 3D effects (reduce particle count by 70%)
- Stack sections vertically
- Hamburger navigation menu
- Touch-friendly button sizes (min 44px)
- Disable parallax on mobile
- Optimize images (WebP format)

### Performance Targets
- Lighthouse score: 90+
- First Contentful Paint: < 1.5s
- Time to Interactive: < 3s
- Reduce 3D complexity on low-end devices
